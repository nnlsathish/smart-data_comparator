User Guide

Step 1: Input Data

Paste Form/Excel data into the left box and Sainpase/CPQ data into the right box. Use the toggle buttons to select if your data has a single header row or multiple header rows.

Step 2: Clean Data

The tool will automatically parse the data. You can review the rows in the preview tables. If a row looks wrong, click the trash icon to delete it.

Step 3: Map Columns

Map the fields from your Form data to the corresponding columns in the Sainpase data. The tool attempts to auto-match names.

Step 4: Results

Review the dashboard. Rows with exact matches are highlighted in green, while mismatches are highlighted in red.
